# **Appium Installation & Setup Documentation**

### **1. Should Appium Server Be Installed in the OS?**  
‚ùå **No**, Appium Server is not installed at the OS level like traditional software.  
‚úÖ It is a **Node.js package** installed via **NPM** or bundled with **Appium Desktop**.

### **2. Is Appium a Node.js Program?**  
‚úÖ **Yes**, Appium is a Node.js program installed using:  
```sh

```
It runs as a **Node.js process**, not a system service.

---

### **3. Where Does Appium Get Installed?**  
- **Windows:** `C:\Users\<YourUser>\AppData\Roaming\npm`  
- **macOS/Linux:** `/usr/local/lib/node_modules/`  

Check Appium‚Äôs install location:  
```sh

which appium  # macOS/Linux

```

---

### **4. Do I Need to Set PATH in Environment Variables?**  
‚úÖ **Usually, No** ‚Äì NPM automatically adds global binaries to `PATH`.  
‚ùå **If `appium -v` fails**, add the following to your `PATH`:  

- **Windows:**  
  ```

  ```
- **macOS/Linux:**  
  ```sh
  export PATH=$PATH:/usr/local/bin:/usr/local/lib/node_modules
  ```

Verify installation:  
```sh
appium -v
```

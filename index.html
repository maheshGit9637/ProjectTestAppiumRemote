Appium Setup Checklist

1. Download and install Node.js from nodejs.org.

2. Verify installation by running:

node -v
npm -v

1. Run the following command:

npm install -g appium

2. Verify installation by running:

appium -v

1. Run the following command:

npm install -g appium-doctor

2. Verify installation by running:

appium-doctor

1. Download and install JDK from Oracle or use OpenJDK.

2. Set the JAVA_HOME environment variable.

3. Verify installation by running:

java -version

1. Download Android Studio from developer.android.com.

2. Install Android SDK tools (e.g., adb, emulator).

3. Set the ANDROID_HOME environment variable.
